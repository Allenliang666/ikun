# 2026-03-27 大六壬金口诀排盘应用开发进度

## 完成功能

### 核心排盘逻辑（已实现）
1. **四柱计算** - 年月日时干支推算
2. **第二步：月将加时推将神** - 月将加到时辰地支，顺数到地分得将神
3. **第三步：依日干起贵神** - 按歌诀确定贵人起点和顺逆方向
   - 甲戊庚：日丑顺/夜未逆
   - 乙己：日子顺/夜申逆
   - 丙丁：日亥顺/夜酉逆
   - 壬癸：日巳逆/夜卯顺
   - 辛：日午逆/夜寅顺
4. **第四步：五子遁立人元** - 日干起子时，顺数到地分地支
5. **命宫计算** - 月支加年支，从年支位置顺数月支位数到时支
6. **十二宫排列** - 从命宫开始按顺逆行排列（阳年男顺/阴年女顺）

### 将神、贵神配干
- 使用五子元遁法为将神和贵神配上天干
- 日干起子时，顺数到将神/贵神地支得配干

### 交互功能
- 点击宫位高亮显示（蓝色边框）
- 下方详情面板显示：人元、贵神、将神、地分、宫位

### 旺衰计算（已实现）
- 五行属性映射（天干、地支）
- 旺的五行判断逻辑：
  - 四位四行：不受克者旺
  - 四位三行：克别人又不被克者旺
  - 四位二行：一比三多者旺 / 二比二主克者旺 / 二比二受生者旺
  - 四位一行：不论旺衰
- 其他五行状态：旺比者旺、旺生者相、生旺者休、克旺者囚、旺克者死
