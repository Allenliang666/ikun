## 干支数据接入

### 核心算法位置
- ikun天文历核心JS在 `sxwnl.htm` 第一个 `<script>` 块（378KB）
- 提取到 `sxwnl/core.js`
- 关键对象：`obb`（全局），关键函数：`obb.mingLiBaZi(jd, J, ob)`
  - `jd`：儒略日减去J2000（即 `jd - 2451545`）
  - `J`：经度弧度（东经120度 = `120 * Math.PI/180`）
  - `ob`：输出对象，包含 `bz_jn`(年柱)、`bz_jy`(月柱)、`bz_jr`(日柱)、`bz_js`(时柱)

### 公历转儒略日公式
```js
var a = Math.floor((14-M)/12);
var yy = Y+4800-a, mm = M+12*a-3;
var jd = (D+0.5) + Math.floor((153*mm+2)/5) + 365*yy
       + Math.floor(yy/4) - Math.floor(yy/100) + Math.floor(yy/400) - 32045 - 0.5;
```
- D 用 `day + 0.5`（正午），不含小时（避免时辰偏差）

### 时柱计算（不用天文历，自算）
```js
var hzIdx = Math.floor((h+1)/2) % 12;  // 时支索引
var hz = Z[hzIdx];
var dayGanIdx = T.indexOf(dgz.charAt(0));
var hStartIdx = [0,2,4,6,8,0,2,4,6,8][dayGanIdx];
var hgIdx = (hStartIdx + hzIdx) % 10;
var hgz = T[hgIdx] + hz;
```

## 六爻排盘（新建）
- 用户需求：报数起卦 + 手动指定卦，时间用ikun天文历
- 目录：`C:\Users\liang\.qclaw\workspace\liuyao\index.html`

- 关键算法：
  - 报数转卦：`sg=(n1%8)||8, xg=(n2%8)||8, dong=(n3-1)%6`

  - 六亲：妻财(克我者)、官鬼(克身)、子孙(我克)、父母(生我)、兄弟(比和)

- JS数据全部用 Python `json.dumps` 生成确保语法正确
- 样式：深色渐变背景 + 金色主题，与其他工具一致
